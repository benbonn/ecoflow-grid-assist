This project is a practical ioBroker JavaScript solution for EcoFlow systems and external smart meters (IR readers).

The goal is **stable 24/7 operation** and **clear, maintainable logic**.

---

## How to contribute

You can help by:

- Reporting bugs (with logs + screenshots)
- Suggesting improvements (small, focused changes)
- Testing on different EcoFlow devices / setups
- Improving documentation (README, diagrams, comments)
- Submitting Pull Requests (PRs)

---

## Where to start

### üêõ Bug reports
Please open a GitHub Issue and include:

1. **Your setup**
   - EcoFlow device(s): (e.g. Stream Ultra / Stream Pro / etc.)
   - ioBroker version
   - `ecoflow-mqtt` adapter version
   - Meter type: IR reader / smart meter / other

2. **Relevant state IDs**
   - `GRID_W`
   - `ECOFLOW_SET`
   - `SOC`
   - `SOC_RES`

3. **Logs (important!)**
   Please paste the relevant `javascript.0` log lines (copy/paste, no screenshots if possible).

4. **What you expected vs what happened**
